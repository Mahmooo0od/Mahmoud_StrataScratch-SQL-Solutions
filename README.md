# Mahmoud_StrataScratch-SQL-Solutions 🚀

Welcome to my repository dedicated to solving real-world SQL interview questions from **StrataScratch**. This project serves as a practice log for mastering complex queries, window functions, and data manipulation techniques.

## 📌 Project Overview
The goal is to solve problems sourced from top-tier companies (FAANG, etc.) to sharpen my data engineering and analytical skills. Each solution is optimized for performance and readability.

## 🛠️ Technologies Used
* **SQL Dialects:** PostgreSQL, MS SQL Server, MySQL.
* **Concepts:** CTEs, Window Functions (RANK, LEAD/LAG), Self-Joins, Subqueries, and Data Aggregations.

## 📂 Repository Structure
Solutions are organized by difficulty level for easy navigation:
- `/Easy`: Basic filtering and simple aggregations.
- `/Medium`: Joins, subqueries, and standard analytical tasks.
- `/Hard`: Complex logic involving multiple window functions and advanced CTEs.
